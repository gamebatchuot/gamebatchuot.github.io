Hướng dẫn thêm âm thanh vào game

Để game có âm thanh, bạn cần thêm 3 file âm thanh vào thư mục này:

1. move.mp3 - Âm thanh khi nhân vật di chuyển. Đề xuất: âm thanh "bước chân" hoặc "vo ve" ngắn.
2. miss.mp3 - Âm thanh khi click hụt. Đề xuất: âm thanh "boing" hoặc "whoosh".
3. catch.mp3 - Âm thanh khi bắt được nhân vật. Đề xuất: âm thanh "ta-da!" hoặc "yay".

Bạn có thể tải các âm thanh miễn phí từ các trang web sau:
- https://freesound.org
- https://mixkit.co/free-sound-effects
- https://pixabay.com/sound-effects

Sau khi tải về, đổi tên file thành move.mp3, miss.mp3 và catch.mp3, rồi đặt vào thư mục này.

Lưu ý: Game vẫn hoạt động bình thường ngay cả khi không có âm thanh.
